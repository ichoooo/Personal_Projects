# RayaMart E-commerce Analysis
- Project Focus: E-commerce Analysis
- Project Kind: Data Analyst Project

**Dataset Info:**
- Title: Messy E-commerce Dataset
- Source: Kaggle
- Link: https://www.kaggle.com/datasets/saidnizam/messy-e-commerce-dataset?select=customers.csv

No. of CSV content: 2

Dataset Problem:

Missing Values - strategic gaps across multiple columns
Duplicate Records - both exact and near-duplicate entries
Outliers - extreme values that will test your detection skills
Data Type Issues - numbers stored as strings, mixed encodings

Created Business Name: RayaMart

Business Question:

How does the customer demographic profile (age, gender, and subscription status) vary across different Indonesian provinces, and where should the marketing team allocate localized campaign budgets to address low engagement?
How long is the operational lag per province before a user makes their first transaction, indicating a need to optimize our regional onboarding funnel? 
What is the total revenue and transaction completion rate across different payment methods (Credit Card, E-wallet, Bank Transfer, Cash) for each Indonesian province, and what common themes appear in the transaction review text for orders that were cancelled or left pending?
