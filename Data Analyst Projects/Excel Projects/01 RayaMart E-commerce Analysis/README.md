# RayaMart E-commerce Analysis

### Project Overview
* **Project Focus:** E-commerce Analysis
* **Project Kind:** Data Analyst Project

### Dataset Info
* **Title:** Messy E-commerce Dataset
* **Source:** Kaggle
* **Link:** [Kaggle Dataset](https://www.kaggle.com/datasets/saidnizam/messy-e-commerce-dataset?select=customers.csv)
* **No. of CSV content:** 2

### Dataset Problems Identified:
* **Missing Values** – strategic gaps across multiple columns.
* **Duplicate Records** – both exact and near-duplicate entries.
* **Outliers** – extreme values that will test your detection skills.
* **Data Type Issues** – numbers stored as strings, mixed encodings.

---

### Business Questions
1. How does the customer demographic profile (age, gender, and subscription status) vary across different Indonesian provinces, and where should the marketing team allocate localized campaign budgets to address low engagement?
2. How long is the operational lag per province before a user makes their first transaction, indicating a need to optimize our regional onboarding funnel?
3. What is the total revenue and transaction completion rate across different payment methods (Credit Card, E-wallet, Bank Transfer, Cash) for each Indonesian province, and what common themes appear in the transaction review text for orders that were cancelled or left pending?
